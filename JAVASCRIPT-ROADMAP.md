## 📍 1. JavaScript Basics  
Before diving into advanced topics, you need a strong foundation.  

### ✅ What is JavaScript?  
JavaScript is a programming language that allows you to add interactivity to your websites. It runs in the browser and powers everything from animations to complex web apps.

### ✅ How Browsers Run JavaScript  
Browsers have JavaScript engines (like Chrome’s V8 or Firefox’s SpiderMonkey) that execute your code.

### ✅ Linking JavaScript to HTML  
You can add JavaScript to your HTML using:  
```html
<script src="script.js"></script>
```

### 📌 Variables & Data Types  
- **Variables**: `var`, `let`, `const`  
- **Data Types**:  
  - Primitive: `String`, `Number`, `Boolean`, `Null`, `Undefined`, `Symbol`  
  - Non-Primitive: `Arrays`, `Objects`

### 📌 Operators  
- Arithmetic: `+`, `-`, `*`, `/`  
- Comparison: `==`, `===`, `!=`, `<`, `>`  
- Logical: `&&`, `||`, `!`

---

## 📍 2. Control Structures  
Control the flow of your program:  
- **If...else**  
- **Switch**  
- Loops: `for`, `while`, `do...while`, `for...of`, `for...in`

---

## 📍 3. Functions in JavaScript  
Functions are the building blocks of JavaScript.  
- **Function Declaration & Expression**  
- **Arrow Functions**  
- **Callback Functions**  
- **IIFE (Immediately Invoked Function Expression)**  
- **Higher-order Functions**

---

## 📍 4. Arrays & Objects  
### Arrays  
- Methods: `push()`, `pop()`, `slice()`, `map()`, `filter()`, `reduce()`  

### Objects  
- Object Literals  
- Accessing Properties  
- `this` Keyword  
- Destructuring, Spread, and Rest Operators

---

## 📍 5. Strings & Numbers  
- String Methods: `length`, `toUpperCase()`, `includes()`  
- Number Methods: `toFixed()`, `parseInt()`, `isNaN()`

---

## 📍 6. DOM Manipulation  
The DOM (Document Object Model) allows JavaScript to interact with HTML elements dynamically.  
- Access Elements: `getElementById()`, `querySelector()`  
- Manipulate Elements: `innerHTML`, `classList`, `style`  
- Create/Remove Elements: `createElement()`, `appendChild()`

---

## 📍 7. Events in JavaScript  
Events make your website interactive.  
- Event Listeners: `addEventListener()`  
- Event Object: `event.target`, `event.preventDefault()`  
- Event Bubbling & Capturing

---

## 📍 8. BOM (Browser Object Model)  
Interact with the browser:  
- `window` Object  
- `alert()`, `prompt()`, `setTimeout()`, `setInterval()`  
- `location.href`, `history.back()`

---

## 📍 9. Error Handling  
Handle errors gracefully:  
- `try`, `catch`, `finally`  
- Throw custom errors

---

## 📍 10. Modern JavaScript (ES6+)  
Stay up-to-date with modern features:  
- `let`, `const`, Arrow Functions  
- Template Literals  
- Destructuring  
- Promises, `async/await`  
- Modules: `import`, `export`

---

## 📍 11. JSON (JavaScript Object Notation)  
Work with APIs:  
- `JSON.stringify()`  
- `JSON.parse()`
